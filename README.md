# 🏨 Hotel Booking and Customer Behavior Analysis

## Project Overview

This project analyzes hotel booking data to uncover customer behavior, booking trends, cancellation patterns, and hotel performance using Python, SQL, Excel, and Power BI.

## Dataset

- Hotel Booking Demand Dataset
- Source: Kaggle

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Excel
- Power BI
